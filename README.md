Sample project showing [AVA](https://github.com/sindresorhus/ava) unit tests for [React](https://facebook.github.io/react/) components using [Enzyme](http://airbnb.io/enzyme).

## Install and try

- Clone this repository
- Run `npm install`
- Run `npm test`
- Enjoy:
